### 840. Range Sum Query - Mutable

###medium

Given an integer array nums, find the sum of the elements between indices i and j (i â‰¤ j), inclusive.

The update(i, val) function modifies nums by updating the element at index i to val.

### Example

```
Given nums = [1, 3, 5]

sumRange(0, 2) -> 9
update(1, 2)
sumRange(0, 2) -> 8
```

### Notice

1.The array is only modifiable by the update function.
2.You may assume the number of calls to update and sumRange function is distributed evenly.

```python
class NumArray(object):
    # Use Binary index tree
    # Remember the code
    def __init__(self, nums):
        """
        :type nums: List[int]
        """
        self.arr = [0 for i in nums]
        self.bittree = [0 for i in range(len(nums)+1)]
        
        for i in range(len(nums)):
            self.update(i, nums[i])
        print(self.arr)
        print(self.bittree)
        
    def update(self, i, val):
        """
        :type i: int
        :type val: int
        :rtype: void
        """
        delta = val - self.arr[i]
        self.arr[i] = val
        index = i + 1
        
        while index <= len(self.arr):
            self.bittree[index - 1] += delta
            index += self.lowbit(index)
        

    def sumRange(self, i, j):
        """
        :type i: int
        :type j: int
        :rtype: int
        """
        return self.get_prefix_sum(j) - self.get_prefix_sum(i-1)
        
    def get_prefix_sum(self, i):
        res = 0
        index = i + 1
        while index > 0:
            res += self.bittree[index - 1]
            index -= self.lowbit(index)
        return res
    
    def lowbit(self, i):
        return i & -i
```
